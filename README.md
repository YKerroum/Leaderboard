# Leaderboard

> The leaderboard website displays scores submitted by different players. It also allows you to submit your score.


## Built With

- HTML/CSS
- JavaScript
- Webpack
- npm
- API
- GitFlow


## Getting Started

Run dist/index.html.

### Prerequisites
This website requires Javascript to be enabled in your browser.
